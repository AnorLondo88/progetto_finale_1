# Progetto finale – Analisi di Vendite in una Catena di Negozi

## Descrizione
Progetto conclusivo del primo modulo del corso Python / Data Science.
L’obiettivo è analizzare i dati di vendita di una catena di negozi di elettronica
utilizzando Python, Pandas, NumPy e Matplotlib.

## Traccia del progetto
Il progetto prevede:
- Creazione di un dataset di vendite (CSV)
- Analisi dei dati con Pandas
- Elaborazioni numeriche con NumPy
- Visualizzazione dei risultati con Matplotlib

## Struttura del progetto
- `progetto_vendite.py` → script principale
- `vendite.csv` → dataset di partenza (generato dallo script)
- `vendite_analizzate.csv` → dataset con colonne e analisi aggiuntive
- `README.md` → descrizione del progetto

## Contenuto dello script
Lo script esegue le seguenti operazioni:

1. Genera un file CSV con dati di vendita simulati
2. Importa il dataset con Pandas
3. Calcola incassi totali e medi
4. Individua i prodotti più venduti
5. Esegue analisi numeriche con NumPy
6. Crea grafici (barre, torta, linee)
7. Raggruppa i prodotti per categoria
8. Salva i risultati in un nuovo CSV
